       || Install DHCP Server in Ubuntu ||

Steps :-
1.	$sudo apt-get update
2.	$sudo apt-get upgrade
3.	Install isc-dhcp-server
$sudo apt-get install isc-dhcp-server
4.	  $sudo vi /etc/default/isc-dhcp-server
5.	$sudo vi /etc/dhcp/dhcpd.conf
6.	$sudo vi /etc/dhcp/dhcpd.conf
7.	$ifconfig
8.	$sudo systemctl restart isc-dhcp-server
9.	$sudo service isc-dhcp-server status
